Problem

Font yang terlalu kecil dan terlalu banya penguraian membuat user merasa cepat jenuh. Permasalahan tersebut membuat user cepat meninggalkan website tanpa membaca semua informasi yang diperlukan mengenai produk.

How I Fix That

Untuk memperbaiki masalah tersebut saya membuat semuat font dengan ukuran diatas rata-rata, sehingga pandangan user langsung tertuju pada content atau informasi yang di sediakan.
