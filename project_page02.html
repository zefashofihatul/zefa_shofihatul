Problem

Font yang terlalu kecil dan terlalu banyak pengurainan membuat user merasa cepat jenuh. Permasalahan tersebut membuat user cepat meninggalkan website tanpa membaca semua informasi yang diperlukan mengenai produk tersebut.

How I Fix That

Untuk memperbaiki masalah tersebut saya membuat setiap halaman website menjadi lebih singkat padat dan jelas, serta saya tambahkan tombol interaksi untuk memindahkan setiap slide halaman-nya.
